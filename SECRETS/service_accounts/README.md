# NAYA — Service Accounts
Place GCP service account JSON files here:
- `naya-cloudrun-sa.json` — Cloud Run deployment
- `naya-storage-sa.json` — Google Cloud Storage
- `naya-bigquery-sa.json` — BigQuery analytics
- `naya-firebase-sa.json` — Firebase / Firestore

**NEVER commit real service account keys to git.**
These files are excluded via `.gitignore`.
